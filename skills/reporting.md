# Reporting Guide

## Purpose

Generate concise Markdown reports summarizing the used Mazda MX-5 market.

## Output Format

Markdown (.md)

Do not generate PDF or HTML unless requested.

## Report Structure

Include:

- Title
- Report date
- Executive Summary
- Market Statistics
- Forecast Summary
- AI Analysis
- Conclusion

## Writing Style

- Professional
- Concise
- Objective
- Data-driven

Avoid marketing language.

## Statistics

Present only calculated values.

Never invent statistics.

Clearly label estimates and forecasts.

## AI Analysis

Summarize:

- Market trend
- Price movement
- Forecast outlook
- Notable observations

Base analysis only on available data.

## Tables

Prefer Markdown tables.

Keep tables readable.

## Charts

Reference generated charts when available.

Do not embed images directly.

## Project Rules

Never:

- exaggerate findings
- invent conclusions
- speculate without evidence
- present forecasts as certainty

Always distinguish:

- Historical facts
- Calculated statistics
- Forecasts
- AI-generated commentary
